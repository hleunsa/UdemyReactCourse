## Indecision App



### Usual commands

- babel compilation and watch
```

babel src/app.js --out-file=public/scripts/app.js --presets=env,react --watch

```
- launch live-server
```
live-server public/
```
- for gitpodio(http://gitpod.io) you may/should add `npx` before the command
